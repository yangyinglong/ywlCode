学生信息包括如下：
```
1. idNumber 学号
2. name 姓名
3. age 年龄
4. sex 性别
5. grade 年级
6. class 班级
7. headmaster 班主任姓名
8. status 信息状态 0 这条信息有效 1 这条信息无效
9. entranceTime 入学时间 201809
10. createTime 信息添加时间
11. updateTime 信息更新时间
```

实现功能：
```
1. 添加学生信息
2. 删除学生信息
3. 查询学生信息
	a. 根据学号查询
	b. 根据姓名查询
	c. 根据班级查询
	d. 根据年级查询
	e. 根据入学时间查询
4. 更新学生信息
	a. 更新学生的年级和班级
	b. 更新学生的年龄
```
