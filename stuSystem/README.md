```
这个文件夹里面就放跟学生系统有关的东西，其他的不要放
```


学生信息包括如下：
```
1. idNumber 学号
2. name 姓名
3. age 年龄
4. sex 性别
5. grade 年级
6. class 班级
7. headmaster 班主任姓名
8. status 信息状态 0 这条信息有效 1 这条信息无效
9. entranceTime 入学时间 201809
10. createTime 信息添加时间
11. updateTime 信息更新时间
```

实现功能：
```
1. 添加学生信息
2. 删除学生信息
3. 查询学生信息
	a. 根据学号查询
	b. 根据姓名查询
	c. 根据班级查询
	d. 根据年级查询
	e. 根据入学时间查询
4. 更新学生信息
	a. 更新学生的年级和班级
	b. 更新学生的年龄
```

具体实现
```
1. 定义学生结构体，表示一个学生信息
	学生信息先只包括 学号，姓名，年龄，信息状态（0：代表这条学生记录被删除，1：代表这条学生记录有效），具体结构体如下;
		struct Student
		{
			char idNumber[6];	// 学号
			char name[10];		// 姓名
			int sex;			// 性别， 1：男， 0：女
			int status;			//学生状态，1：有效，0：无效
		};
	后续对文件操作熟悉了，再加上后面的字段

	int main()

		// 根据性别查询符合条件的所有学生的信息并打印出来(f)，打印所有的学生信息(p)，退出系统(q)）。
```
